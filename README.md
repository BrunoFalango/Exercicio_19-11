Uma asa é no seu sentido mais comum, um conjunto de paredes, cômodos e teto construídos pelo ser humano com a finalidade de constituir um espaço de habitação para um indivíduo ou conjunto de indivíduos para que estejam protegidos dos fenômenos naturais exteriores (como a chuva, o vento, calor e frio etc.), além de servir de refúgio contra ataques de terceiros
